Como usar este dashboard

Abre este archivo con un servidor local. Desde la carpeta del proyecto ejecuta: python -m http.server 8080 y luego abre http://localhost:8080/dashboard.html en tu navegador. El dashboard cargara automaticamente los datos del ciclo mas reciente.
